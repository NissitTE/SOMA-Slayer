
**SOMA SLAYER** is a fast-paced, web-based survival RPG built for the *Somagamasu* project. Battle through endless floors, collect rare loot, and unlock powerful hidden classes in a dark, neon-lit dungeon atmosphere.

### 🎮 How to Play
- **WASD / Arrow Keys**: Move your character.
- **Mouse**: Aim and Attack.
- **Numbers 1-2**: Use consumables (HP/MP Potions).
- **Q / E / R**: Skill slots (varies by class).
- **D**: Toggle Debug Console (Added in v11.0).
- **ESC**: Pause / Menu.

### ✨ Key Features
- **Progressive Difficulty**: Enemies scale as you descend deeper into the floors.
- **Rarity System**: Loot drops range from *Junk* to the mythical *SOMA* tier.
- **Hidden Classes**: Unlock advanced roles like the *Excalibur Knight* or *.50 Calibre Archer*.
- **Dynamic Obstacles**: Randomized dungeon layouts for every run.

*Developed as part of the Somagamasu (โซมากามาซุ) Web Project.*
