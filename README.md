
**SOMA SLAYER** is a fast-paced, web-based survival RPG built for the *Somagamasu* project. Battle through endless floors, collect rare loot, and unlock powerful hidden classes in a dark, neon-lit dungeon atmosphere.

SOMA SLAYER is a web-based RPG dungeon crawler where strategy meets action. Progress
through floors, upgrade your attributes, and collect the legendary SOMA gear.
Class Overview
Knight: Defensive specialist (VIT focus). Secret Class: Excalibur.
Archer: Speed & Precision (AGI focus). Secret Class: .50 Calibre.
Priest: Magical Arts (INT focus). Secret Class: Merlin Bloodline.
Core Systems
Mastery: Skills scale up to Lv.99 with +2% damage per level.
Forging: Use Diamonds for "Heavenly" options or recycle Junk for higher-tier equipment.
Dungeon Navigation: Dynamic map generation with floor selection and boss toggles.

Release Notes v12.1

Major Changes:
Find Set Equipments that give player +3% DEF +++
Jewel Forge: Added custom stat re-rolling using Diamonds.
Smart Re-Forge: Mass-recycle items while automatically preserving better gear.
Junk Forge x100: High-speed recycling for veteran players.
UI Enhancements: New sorting/filtering in inventory and improved hotkey mapping.
Balance: Optimized skill damage scaling and adjusted mid-game boss health.
